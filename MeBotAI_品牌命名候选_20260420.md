# MeBotAI 品牌命名候选
> 整理日期：2026-04-20 · 批次 01
> 市场优先级：中文市场优先

---

## 产品定位（命名的出发点）

- 私有信息定价与变现平台
- 博主的智慧在问答中沉淀，粉丝付费访问
- 核心立意：**卖的是智慧，不是知识**。知识可以搜索，智慧只能请教
- 底层哲学：苏格拉底从不著书，所有智慧在对话和问答中产生
- 核心价值：你买不起五个顾问，但你负担得起五个知识库

---

## 当前首选候选

| 名称 | 释义 | 延展性 |
|------|------|--------|
| **灼知** | 「灼」= 穿透力/照亮，「知」= 真正懂得（比「见」更深一层）。有「被一句话点透」的感觉，直接区分「智慧≠知识」 | 灼知 XX / 问灼知 / 一灼知 |

**相关候选（同系列，备用）**：

| 名称 | 释义 |
|------|------|
| **灼问** | 灼热的追问，直指本质的提问 |
| **灼语** | 有分量、能照亮的话 |
| **知微** | 「见微知著」，洞察细节背后的规律 |
| **知几** | 《易经》「知几其神乎」，「几」= 事物将变未变的关键时机，知几 = 在关键处有洞察 |

| 名称 | 排除原因 |
|------|---------|
| 一席 | 已被 2012 年知名演讲平台占用（中国版 TED） |
| 问道 | 已被雷霆网络现象级手游占用，流量极大 |
| 问津 | 已被长亭科技安全大模型占用，另有同名知识交易软件 |
| 柏拉图 | 认知已被「柏拉图式恋爱」占据，品牌定位模糊 |
| Delphi | 多个 AI 产品已占用 |
| Agora | 偏国际化，中文市场优先故降级 |

### 二、备选池（未被大品牌占用，保留参考）

| 名称 | 释义 | 备注 |
|------|------|------|
| **叩问** | 敲门求教，「叩」天然带付费门槛的意象 | 文学质感强，稍显郑重 |
| **请益** | 《论语》原词，字面意思即「郑重付费请人指教」，几乎为产品量身定做 | 部分用户不识此词 |
| **稷下** | 战国齐国稷下学宫，百家争鸣之地，多博主多视角的场景 | 中文历史感强 |
| **追问** | 不满足于第一个答案，持续深挖，最贴苏格拉底精神 | 现代感好，仅作知乎功能名，未独立成品牌 |
| **鹅湖** | 朱熹与陆九渊「鹅湖之会」，两派哲人面对面辩论，中国哲学史最著名的思想碰撞 | 画面感强，冷门但干净 |
| **一问** | 一个问题解决一个问题，极简 | 偏工具感，缺哲学深度 |
| **小语录** | 语录文化，《论语》《传习录》的轻量化版 | 亲切接地气，偏静态 |

### 三、苏格拉底产婆术方向（国际化备用）

| 名称 | 释义 | 备注 |
|------|------|------|
| **Maia / 迈雅** | 苏格拉底「产婆术」（Maieutics）词根，希腊助产女神。博主的 AI 不是替你思考，而是帮你「生出」自己的判断 | 中英双栖，国际化优先时首选 |
| **Phron** | 亚里士多德 Phronesis（实践智慧）缩写，专门区分「智慧≠知识」的哲学术语 | 学术精英气浓 |
