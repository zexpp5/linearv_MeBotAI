# Part I · 分析框架

## 1.1 第一性原理：三层过滤器

筛博主的第一性工具。**不用这个筛，任何赛道都会失控。**

| 类型 | 定义 | 代表赛道 |
|---|---|---|
| **地理位置型** (Location-bound) | 信息的获取成本是"在那个地方待过 N 年" | 地方房产、开店选址、地方餐饮、产业带 |
| **关系网络型** (Relationship-bound) | 信息的获取成本是"认识那个圈子的 N 个人" | 地方律师 + 法官圈子、医疗就医、政府招投标 |
| **案例积累型** (Case-history-bound) | 信息的获取成本是"亲历 N 个案例" | 保险理赔、二手车验车、判例、医美术后 |

**核心判断规则：一个博主必须至少命中两层过滤器，才值得深挖。**

理由：ChatGPT 不可能"在徐州鼓楼区待了 12 年"、不可能"和李法官喝过 300 顿酒"、不可能"亲手验过 800 台二手车"。三层积累只能由**物理 / 社会 / 时间**取得，不是智力取得。**这才是真正的护城河。**

## 1.2 三个必要条件 + 两个加分条件 + 两个强排除

筛博主的完整判断清单：

### 必要条件（全部满足才进候选）
1. ✅ **物理 / 本地 / 行业内部** 的一手信息（ChatGPT 抓不到）
2. ✅ **粉丝的问题是可问答的**（"这个楼盘能买吗"、"义乌验质怎么做"），不是情绪型（"今天好难该怎么办"）
3. ✅ **决策金额大或后果重**（决定粉丝愿不愿付 9.9 以上）

### 加分条件（命中越多越好）
4. ➕ 博主时间受限（粉丝数 >> 博主能 1v1 答的数量）→ 天然需要 AI 杠杆
5. ➕ 博主已经在卖某种付费产品（说明接受了"我是付费型博主"的身份转换）

### 强排除（命中任一 = 自动淘汰）
6. ❌ 赛道高度同质化（升学、副业加盟、泛留学）
7. ❌ 合规重压（诊疗、开药、具体法律意见、具体投资推荐）—— 但可做边缘科普

## 1.3 产品适配度维度

即便赛道选对了，**产品能不能发挥最大价值**还要看一个独立维度：**Q&A 结构化程度**。

| 赛道类型 | 问答结构化程度 | 例 |
|---|---|---|
| ✅ 高度 Q&A 化 | 问题有具体、可验证、可存储的答案 | "协和胸外科哪个教授号好挂" |
| ⚠️ 中度 Q&A 化 | 问题有答案但需要情境判断 | "XX 店铺选在 XX 路段如何" |
| ❌ 叙事性 / 策略性 | 问题需要听完一整套故事或持续跟进 | "我该怎么做自媒体"、"明年该怎么投资" |

**问诊就诊赛道在这个维度上分数最高** —— 这是它被放入第一档的额外理由。

## 1.4 选择悖论识别法

**遇到"教 X"类赛道必用的一步检查。**

> 如果他教人做 X 做得这么成功，为什么要花时间来教？

### 悖论有两种：**强悖论（直接淘汰）** vs **弱悖论（需要额外筛选）**

#### 强悖论 = 教学即产品，循环自指 → 直接淘汰

- **教自媒体、教拍视频、教 IP 运营** —— 他教"自媒体"，他的产品就是"教自媒体的自媒体"。循环。
- **教创业、教赚钱、教副业** —— 他教赚钱，教本身就是他的赚钱方式。循环。
- **泛"教人找工作"** —— 他如果真那么会找工作，为什么坐在这里教别人？循环。
- **泛简历指导** —— 同上。

#### 弱悖论 = 技能可迁移，当前产品 ≠ 原来技能 → 需要额外筛选

- **前字节 P7 教你拿字节 offer** —— 他原来的技能是"做字节 P7 的工作"，现在的产品是"教学"。两者是不同技能，可以并存。不是悖论。
- **前麦肯锡顾问教你拿咨询 offer** —— 同上。
- **前骨科医生教你选骨科医生** —— 同上。
- **前开发商销售经理教你买房** —— 同上。

**弱悖论型博主需要额外的可信度筛选（三选一必须命中）：**
1. 可验证的前职履历（领英 / 招聘网站 / 公司官网）
2. 粉丝成果案例（真实 offer 截图 / 录取截图）
3. 业内人交叉背书（其他从业者承认他是业内人）

### 典型归类

| 赛道 | 悖论类型 | 处理 |
|---|---|---|
| 教自媒体 / 拍视频 | 强悖论 | 淘汰 |
| 教创业 / 教赚钱 / 副业 | 强悖论 | 淘汰 |
| 泛简历指导 / 泛找工作 | 强悖论 | 淘汰 |
| 前大厂员工教大厂求职 | 弱悖论 | 保留 + 筛可信度 |
| 前投行教投行面试 | 弱悖论 | 保留 + 筛可信度 |
| 医生 / 律师 / 中介教用户 | 无悖论 | 保留 |
| 教就医路径 / 学区房 / 选址 | 无悖论 | 保留 |

## 1.5 本次分析的方法论（如何得到这份清单）

1. **穷举收集** — 从创始人 PMF 文档 + 商业分析文档 + 实际市场扫描 + 随机观察，列出所有候选赛道
2. **三层过滤器打分** — 每个候选按"地理 / 关系 / 案例"三层打分（0 / ✅ / ✅✅）
3. **加分项 + 排除项检查** — 应用必要条件、加分条件、强排除
4. **产品适配度评估** — 每个保留赛道按 Q&A 结构化程度打分
5. **商业成熟度评估** — 每个赛道是否已有博主收付费（已付费 vs 未付费）
6. **冷启动难度评估** — 从 C 端用户身份接触博主的成本 + 信任建立周期
7. **综合排序 + 时机分层** — 第一档 / 第二档 / 第三档 / 备选 / 淘汰

## 2.2 第一档详解（Week 1 主攻 3 个）

**第一档的共同特征：** 三层过滤器至少 2 层命中 + Q&A 适配度高 + 商业成熟度中到高 + 冷启动难度低（博主容易通过"付费咨询"直接接触）。

### 第一档 · ① 房地产中介（地方）

**为什么在第一档：**
- **三层命中**：地理 ✅✅（深度本地知识）+ 关系 ✅（中介 + 银行 + 开发商人脉）+ 案例 ✅（经手大量交易）
- **Q&A 适配度**：高 ——"XX 学区能买吗"、"XX 小区户型优劣"、"XX 月供"都是清晰问答
- **商业成熟度**：高 —— 大量本地房产博主已经挂"付费咨询"
- **决策金额**：家庭最大一笔决策（100 万+），粉丝付费意愿天花板高
- **冷启动**：从"我是买家，付费咨询"切入最自然

**为什么 Week 1 做它：** 创始人在对话中对这一类有直觉，Week 1 用直觉最强的赛道启动能保持动力。

### 第一档 · ② 问诊就诊攻略 🔥

**为什么在第一档（也是三档中最强的）：**
- **三层命中**：地理 ✅（具体医院流程）+ 关系 ✅✅（科室关系 + 医生熟人）+ 案例 ✅（陪过 N 个病例）
- **Q&A 适配度**：**极高** —— 医疗导航问题比其他赛道都更 Q&A 化
- **商业成熟度**：中 —— 陪诊师职业近 2 年兴起，已有付费业务（300-800/次）
- **决策重要性**：绝望型付费，生死攸关
- **产品适配独特性**：博主经验可以结构化成一个真正能被问答的知识库 → AI 答疑 + 引用原文 + 置信度降级的产品设计能发挥**最大价值**

**为什么 Week 1 做它：** 这是 MeBotAI 产品最适配的赛道。如果这个都跑不通，说明产品路线有根本问题，早发现早修正。

**法律边界必须记住：** 只做 **陪诊 + 就医流程 + 医保政策 + 跨省就医**。永远不做诊疗建议 / 开药 / 诊断。

### 第一档 · ③ 开店 / 选址（本地）

**为什么在第一档：**
- **三层命中**：地理 ✅✅（街道级知识）+ 关系 ✅（商场招商 + 业主 + 同行）+ 案例 ✅（3+ 家店主 / 50+ 咨询案例）
- **Q&A 适配度**：高 ——"XX 路段适合开奶茶店吗"、"XX 商场租金值吗"都是清晰问答
- **商业成熟度**：高 —— 选址咨询、加盟避坑咨询已有成熟付费市场
- **历史联系**：这是创始人 PMF 文档里的原始场景（奶茶店选址）—— 回到原点验证

**为什么 Week 1 做它：** 场景最具象、粉丝问题最明确、和创始人原始直觉高度一致。

## 2.3 第二档详解（Week 2-3 扩展 10 个）

**第二档的共同特征：** 赛道本身优质（三层过滤器命中强），但有一项限制需要 Week 1 先解决 —— 要么 Q&A 适配度中等、要么冷启动门槛高、要么合规边界模糊、要么创始人 Wizard of Oz 难度高。

### 第二档 · ① 求职陪跑（细分：行业 / 大厂 / 投行 / 考公）🌟 第二档顶端

**五个 STRONG 子赛道：**

| 子赛道 | 代表博主类型 | 客单价 | 商业成熟度 |
|---|---|---|---|
| **行业求职陪跑（🆕 最强子赛道）** | 前医药员工教药企求职 / 前建筑师教设计院求职 / 前金融民工教券商求职 | 299-999/次 | ⭐⭐⭐⭐⭐ |
| **互联网大厂求职** | 前字节 / 腾讯 / 阿里 P7+ | 500-3000/次 | ⭐⭐⭐⭐⭐ |
| **投行 / 咨询求职** | 前 MBB / 前 IBD / 前外资投行 | 1000-5000/次 | ⭐⭐⭐⭐⭐ |
| **考公 / 选调生（地方）** | 近 2 年上岸的本省人 | 300-2000/次 | ⭐⭐⭐⭐ |
| **公司内部真实体验分享** | 某公司某岗位 2-5 年员工 | 99-499/次 | ⭐⭐⭐⭐ |

**🎯 核心洞察：行业求职陪跑 = "次代职友圈"**

以前用户去**职友圈 / 看准网 / 应届生论坛**查公司内幕。这些平台现在衰落的原因：
- 信息过时（3 年前的薪资参考无效）
- 无策展（匿名水军 + 离职者泄愤污染内容）
- 无行业区分（医药 / 金融 / 建筑求职逻辑完全不同，但平台混在一起）

**用户现在真正要的 = "一个真正懂这个行业的前从业者 + 持续更新的独家内幕"。**

这恰好是 MeBotAI 能通过 AI 放大的对象：**博主把自己的行业知识 + 公司黑名单 + HR 关系 结构化成知识库，粉丝付费问答，博主从 1v1 的时间瓶颈解放出来。**

**📊 真实案例数据点（2026-04-20 抖音扫描）：**

**抖音账号：药企在逃打工人**
- 粉丝 likes: 2,296（典型小 V，流量弱）
- 橱窗主打：**医药成长信息圈 找工作 写简历 过面试**
- 产品配置：**2 年 1V1 咨询 + 行业报告 + 链接资深 HR + 简历/PPT 模板 + 独家技术题库 + 医药求职系列 + 独角兽企业岗位更新**
- 定价：299 元（原价 599，满减 400）
- 销量：**119 份已售 → 单品收入 35,000+ 元**

**这一个博主就验证了：**
1. ✅ 小流量 + 高付费意愿的"信息壁垒高、流量弱"画像真实存在
2. ✅ 医药行业 insider 求职知识库有付费市场
3. ✅ "2 年 1V1 咨询"是典型的时间瓶颈型产品 —— 正是 MeBotAI 切入点
4. ✅ 299 元 × 119 销量 → 简单数学说明这个赛道能养活博主

**三层命中：**
- 地理 ⚠️ —— 互联网 / 投行不具地理壁垒，但考公是 ✅✅（每省考法、面试、晋升都不一样）
- 关系 ✅✅ —— 内推资源 / 面试官熟人 / 业内人脉
- 案例 ✅✅ —— 亲历 N 个 offer、面试场次、部门/岗位细节

**为什么不进第一档：**
1. **Q&A 适配度中等** —— 求职问题很多是情境判断（"我这个背景能进 XX 公司吗"），不是纯知识查找。
2. **Wizard of Oz 阶段你代答困难** —— 你不是前字节 P7，不能假装给出内部工资结构、面试流程、晋升路径。这个赛道博主的价值就在"**他亲历过**"，你无法人肉替代。
3. **选择悖论需要筛选（弱悖论）** —— 不是直接淘汰，但需验证博主前职履历真实性。

**为什么值得第二档顶端：**

## 2.6 明确淘汰赛道及理由

### 淘汰类别 1：三层过滤器 0 命中（无信息壁垒）

- **教 AI 使用** — 工具知识全网共享。任何人读完 Anthropic 文档都能教
- **泛情感咨询** — 不可问答化 + 无本地壁垒
- **泛心理咨询** — 合规边界 + 无本地壁垒（但"特殊儿童康复"可以做，属于案例型）

### 淘汰类别 2：强选择悖论（产品自指）

- **教自媒体 / 拍视频** — 如果他自媒体真成功，为什么教？悖论。
- **教创业 / 副业 / 赚钱** — 自指 + 90% 是加盟/割韭菜
- **教人做 IP** — 自指
- **泛简历指导 / 泛"教人找工作"** — "如果他真那么会找工作，为什么现在坐在这里教别人？"
- **MCN 包装的"爆款训练营" / "一人公司训练营"** — 教别人复制自己的包装手法，典型循环

### 🚫 反面教材案例：抖音"泥巴老师"（2026-04-20 扫描）

**账号信息：** 3,963 粉 + 15.6K 赞 + MCN 创始人 + 全网粉丝 20w + 多个万粉操盘手。

**橱窗产品：**
- 自媒体爆款训练营 **¥699**
- 从 0-1 打造能赚钱的一人公司 **¥299**
- 从 0-1 打造能赚钱的一人公司 **¥1980**（高价版）

**为什么是反面教材 —— 三个产品全部命中强选择悖论：**

1. **"自媒体爆款训练营"** → 她的产品就是自媒体，教的是如何打造自己这种自媒体 → 循环。
2. **"从 0-1 打造能赚钱的一人公司 ¥299"** + "**¥1980**" → 如果她一人公司真从 0-1 打造了能赚钱，为什么做的主业是教别人？**教学就是她赚钱的那个"一人公司"** → 闭环。
3. MCN 包装信号 + "多个万粉操盘手" → 典型 MCN 矩阵式教学。

**为什么我们明确不服务这种博主：**
- 她的粉丝**不是为"真实行业内幕"付费**，是为"我也能复制她的成功"的幻觉付费
- 她的"知识库"内容是包装技巧 + 流量套路，不是物理 / 关系 / 案例积累
- 她没有我们能放大的**独一无二的信息壁垒**
- 粉丝质量低（很多是想"被她教会赚钱"但永远赚不到的人）

**判断规则：** 看一个博主的橱窗里**三个以上产品**都是"教学如何成功"，而不是"分享行业知识" —— pass。

- **纯观点输出型大 V** — 产品适配度 0

## 2.7 知识星球作为对标平台与靶库

### 为什么知识星球是 MeBotAI 的直接前身

知识星球的模式 = **创作者建圈 + 粉丝付费看内容**。本质和 MeBotAI 做同一件事，只有一个关键差别：

| 维度 | 知识星球 | MeBotAI |
|---|---|---|
| 交互模式 | **被动**（博主发帖 → 粉丝读）| **主动**（粉丝问 → AI 从博主内容答）|
| 博主时间依赖 | 高（发帖 + 回复）| 低（内容一次入库 + AI 代答）|
| 粉丝等待时间 | 博主有空时（常常 24h+）| 实时 |
| 升级漏斗 | 无系统化 | 低价 → 高价自动触发 |

**结论：知识星球证明了这个市场真实存在 + 粉丝愿意付费 + 博主愿意建圈。它也暴露了自己的短板（被动 / 慢 / 无漏斗）—— 这就是 MeBotAI 的 insertion point。**

### 每一个现有知识星球博主都是潜在的 Plan A 目标

已经在知识星球运营的博主具备以下信号：
- ✅ 已接受"付费型博主"身份（最难的心态转换已完成）
- ✅ 已把知识结构化（发过几十上百帖）

## 3.4 通用赛道扫描 SOP（每赛道 1-1.5 小时）

### Step 1：名单（20 分钟）
- 上新榜 / 飞瓜 / 蝉妈妈（优先新榜），搜赛道关键词
- 拉 50-100 名粉丝量 **1k-10w** 的腰部博主
- 筛掉明显卖货型 / 纯流量型 / MCN 机构号
- 剩 20-30 个"看起来像真实从业者"

### Step 2：评论区扫描（30 分钟）
- 每个博主看 2-3 个最高互动视频
- **读评论区**（不是视频内容）
- 数：评论是"高频重复问题"（好）还是"点赞情绪"（差）
- 摘记 5-10 条典型真实问题 —— 这是你未来 demo 的语料

### Step 3：变现行为扫描（10 分钟）
- 他是否挂了"1v1 咨询" / "付费群" / "知识星球" / "私信咨询"
- **已有付费 = 他接受了"付费型博主"身份，冷启动更容易**
- 没有付费 = 要教育他这件事（难一档）
- 无任何变现入口 = 可能是小而精专家，你是他第一个商业化伙伴

### Step 4：3 个冷启动候选 + 1 个本周触达（30 分钟）
- 每个赛道选出 3 个最匹配的博主
- 其中至少 1 个，**本周加微信或私信**
- 话术不是"我要跟你合作"，是"**我是你的粉丝，有个问题想付费问你**" —— 做 C 端用户再谈 B 端

## 3.5 30 秒真假专家识别法

打开一个主页看 3 样：

1. **评论区问题的具体度**
   - ✅ "XX 型号好不好"、"XX 地点能去吗" = 真专家
   - ❌ "太棒了！" "老师牛！" = 流量博主

2. **视频内容的颗粒度**
   - ✅ 讲到具体楼盘 / 店铺 / 型号 / 条款
   - ❌ 讲到宏观趋势 / 行业未来

3. **变现链路**
   - ✅ 挂"咨询" / "知识星球" / "1v1" 入口
   - ⚠️ 只挂"直播"或"课程购买"（流量型，合作门槛高）
   - ✅ 无任何变现入口（可能是小而精专家 → 你是第一个商业化伙伴）

- 粉丝粘性极强（通勤听、长时间投入）
- 变现渠道初级 —— **我们可能是他们第一次真正商业化的机会**

**搜索关键词：**
- 行业 + 深度 / 内部 / 复盘 / 老兵
- 具体行业名：医疗 / 教育 / 投资 / 法律 / 建筑 / 餐饮 / 保险 / 外贸

**重点看：**
- 小宇宙"精选" 里垂直行业类播客
- 博主本身是行业内从业者（不是媒体人访谈）
- 有付费社群 / 知识星球入口的

**排除：**
- 综合型大播客（日谈公园、故事 FM 等）
- 媒体人 / 记者访谈型
- 纯内容消费型（书评 / 影评）

**产出：10 + 播客博主名单（附节目链接 + 简介）**

## 决策记录（本文档产出的关键判断）

1. **三层过滤器**作为第一性筛选原理（Part I.1.1）—— 可持续用于未来所有赛道评估
2. **选择悖论识别法**（Part I.1.4）—— 强悖论 vs 弱悖论的区分，决定是否淘汰
3. **Q&A 结构化程度**作为产品适配度维度（Part I.1.3）—— 解释为什么问诊赛道是第一档
4. **Week 1 三赛道确定**：房地产（地方）+ 问诊就诊攻略 + 开店选址
5. **博主触达方式确定**：以 C 端付费体验身份接触，而不是 B 端合作话术
6. **核心验证问题**："你有没有因为时间不够推掉过咨询？" —— 一句话鉴别目标客户
7. **求职陪跑放第二档顶端**：大厂 / 投行 / 咨询 / 考公子赛道商业成熟度极高，但 Q&A 适配度中等 + Wizard of Oz 代答难度高，暂不进 Week 1
8. **考公 / 选调生是求职赛道里地理壁垒最强的子赛道** —— 如果 Week 1 节奏轻松，可周四晚加扫本地考公赛道
9. **泛简历指导明确淘汰**：触发强选择悖论 + ChatGPT 已充分替代
10. **知识星球是 MeBotAI 的"被动前身"**（2.7 节）—— 同一商业模式，只有"被动 vs 主动"差别。已付费运营的星球主 = MeBotAI 冷启动**最高质量靶库**
11. **新增 3 个赛道（来自知识星球扫描）**：
    - **专业技术行业老兵**（失效分析 / EHS / 设计院 / 油运等 → 第二档顶端）
    - **B 端企业咨询顾问**（运营商政企 / 数据化管理 → 第二档）
    - **信用卡 / 点数 / 里程 / 酒店会籍**（硬核省钱变现 → 第二档）
12. **合并确认第二档顶端 3 强**：求职陪跑 + 专业技术行业老兵 + 地方律师判例 —— Week 2 启动时从这 3 个里选 1-2 个
13. **周六新任务：多平台靶库扫描**（Part IV.4.4）—— 知识星球 / 抖音橱窗 / 小红书橱窗 / 在行 / 播客，产出 **80-100 个候选博主的总靶库**，供 Week 2 冷启动使用
14. **在行平台是最精准的单点来源** —— 平台上所有人都已经是付费博主，身份转换已完成，对我们而言搜索成本最低
15. **播客博主是未充分商业化的宝藏** —— 已有结构化内容（每期一主题）、粉丝粘性极高、多数没有深度变现，可能是 MeBotAI 的理想冷启动对象
16. **行业求职陪跑是求职赛道的最强子细分**（2026-04-20 新增）—— 医药、建筑、金融等行业的前从业者教行业求职。三层全命中（关系 + 案例），技能可迁移不触发悖论，粉丝画像非常精准。案例：抖音"药企在逃打工人" 299 元 × 119 销 = 35k+ 收入。
17. **MeBotAI 的一个定位语：**"**次代职友圈**" —— 职友圈 / 看准网衰落（信息过时、无策展、被水军污染），用户现在要的是"一个真正懂行业的前从业者 + 持续更新的独家内幕"，这正是我们用 AI 能放大的对象
18. **反面案例识别规则**（2026-04-20 新增）—— 看橱窗里是否 3 个以上产品都是"教学如何成功"而非"分享行业知识"，是则 pass。典型：MCN 包装账号 + 爆款训练营 + 一人公司训练营 = 强选择悖论组合拳
